## Główne projekty

### [GameCollectionApp](https://github.com/konradprojects/game-collection-app)

Aplikacja desktopowa do zarządzania kolekcją gier, wykonana w języku C# z wykorzystaniem Avalonia UI.

Projekt wykorzystuje architekturę MVVM, Entity Framework Core oraz bazę danych SQL Server. Aplikacja umożliwia:

- wyświetlanie kolekcji gier,
- dodawanie nowych rekordów,
- edytowanie i usuwanie danych,
- filtrowanie gier,
- obliczanie statystyk kolekcji,
- wykonywanie asynchronicznych operacji bazodanowych.

W projekcie zastosowałem również migracje Entity Framework Core, własną kontrolkę użytkownika oraz konfigurację połączenia z bazą danych za pomocą zmiennej środowiskowej.

### [Baza danych wyników Formuły 1](https://github.com/konradprojects/f1-sql-server-baza-danych)

Projekt relacyjnej bazy danych dotyczącej wyników Formuły 1, wykonany w Microsoft SQL Server.

Projekt obejmuje:

- zaprojektowanie struktury bazy danych,
- utworzenie tabel, kluczy podstawowych i obcych,
- zastosowanie relacji i ograniczeń integralności danych,
- przygotowanie danych testowych,
- wykonanie operacji `INSERT`, `SELECT`, `UPDATE` i `DELETE`,
- przygotowanie zapytań analitycznych.

W zapytaniach wykorzystałem między innymi:

- `JOIN`,
- `GROUP BY`,
- funkcje agregujące,
- `CASE`,
- CTE,
- `PIVOT`,
- `ROLLUP`,
- `CUBE`,
- `RANK()`,
- `NTILE()`.

Projekt umożliwia analizowanie klasyfikacji kierowców i zespołów, liczby zwycięstw, miejsc na podium, wyników DNF oraz zmian pozycji pomiędzy startem i metą.
