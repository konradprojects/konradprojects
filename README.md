## Główne projekty

### [GameCollectionApp](https://github.com/konradprojects/game-collection-app)

Aplikacja desktopowa do zarządzania kolekcją gier, wykonana w C# i Avalonia UI.

Projekt wykorzystuje architekturę MVVM, Entity Framework Core oraz bazę danych SQL Server. Aplikacja umożliwia dodawanie, edytowanie, usuwanie i filtrowanie gier oraz obliczanie statystyk kolekcji.

### [Baza danych wyników Formuły 1](https://github.com/konradprojects/f1-sql-server-baza-danych)

Relacyjna baza danych wykonana w Microsoft SQL Server.

Projekt obejmuje tworzenie tabel i relacji, dane testowe, operacje CRUD oraz zapytania analityczne wykorzystujące między innymi `JOIN`, `GROUP BY`, CTE, `PIVOT`, `ROLLUP`, `CUBE`, `RANK()` i `NTILE()`.
